微博数据爬取项目

注意: 这是一个Python后端项目，不能在前端托管平台上运行。此页面仅用于展示项目信息。

项目描述

这是一个用于爬取微博数据的Python项目，可以采集特定关键词在指定时间段内的微博内容及用户信息。

功能特性

  • 爬取微博内容、点赞数、评论数、转发数
  • 获取用户信息：昵称、性别、签名、认证状态
  • 支持时间段筛选（2008-2019年）
  • 生产者消费者模式，支持多线程爬取
  • 数据存储到MySQL数据库

运行要求

  • Python 3.6+
  • MySQL数据库
  • 安装依赖: pip install -r requirements.txt

使用方法

# 1. 安装依赖
pip install -r requirements.txt -i https://pypi.douban.com/simple

# 2. 配置数据库
# 编辑 config.py 文件，设置MySQL连接信息

# 3. 创建数据库表
python3 model.py

# 4. 创建爬取任务
python3 create_task.py

# 5. 开始爬取
python3 search_by_db.py

注意: 此项目需要在支持Python的服务器环境中运行，不适合部署到静态网站托管平台。
